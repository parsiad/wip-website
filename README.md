[Zola](https://getzola.org) blog that uses Jupytext for blog posts.

## Dependencies

* `jupytext`
* `jupyter nbconvert`
* `watchexec`

## Usage

Create blog posts by making Jupytext files in `src/`.

You can make a debug server by running

```
./serve
```

You can build the static website by running

```
./build
```
